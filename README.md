# 🛡️ CodeGuard Scanner

A web-based code security scanner built using HTML, CSS, and JavaScript. This project provides a clean interface for scanning code snippets and demonstrates front-end web development skills.

## 🚀 Live Demo
🔗 https://jiveshai-07.github.io/codeguard-web-scanner/

## 📌 Features
- User-friendly interface
- Responsive design
- Code scanning interface
- Fast and lightweight

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
