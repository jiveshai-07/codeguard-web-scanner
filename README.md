# 🛡️ CodeGuard Scanner

A lightweight web-based security scanner that analyzes JavaScript code for common security vulnerabilities and provides instant recommendations to help developers write safer and more secure code.

---

## 🌐 Live Demo

🔗 https://jiveshai-07.github.io/codeguard-web-scanner/

---

## 📖 Project Overview

CodeGuard Scanner is a browser-based application built using HTML, CSS, and JavaScript. It helps identify insecure JavaScript coding practices and encourages developers to follow secure coding standards. The project demonstrates frontend development, JavaScript programming, and basic security concepts.

---

## ✨ Features

- 🔍 Detects common JavaScript security issues
- ⚡ Instant code analysis
- 🎨 Clean and responsive user interface
- 📱 Mobile-friendly design
- 🚀 Fast browser-based execution
- 💻 No installation required

---

## 🛠️ Technologies Used
